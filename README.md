# Transcribe-Whisper-2026

# Watcher Software README

## Whisper Watcher — Automatic Audio Transcription Monitor

A lightweight Python-based transcription watcher that continuously monitors a folder for new audio/video files and automatically transcribes them using Faster Whisper. Built for simple local-first workflows with minimal setup. 

---

# Features

* Automatically watches a folder for new media files
* Supports:

  * `.mp3`
  * `.m4a`
  * `.wav`
  * `.mp4`
  * `.ogg`
  * `.flac`
* Uses Faster Whisper for fast local transcription
* Saves transcripts automatically as `.txt`
* Skips already-transcribed files
* Continuous polling watcher (hands-free workflow)
* Fully local execution (no cloud APIs required)

---

# Folder Structure

The software automatically creates the following folders on your Desktop:

```text
Desktop/
└── Transcribe Whisper 2026/
    ├── Input/
    └── Output/
```

## Input Folder

Place audio or video files here:

```text
Desktop/Transcribe Whisper 2026/Input
```

## Output Folder

Completed transcripts are saved here:

```text
Desktop/Transcribe Whisper 2026/Output
```

---

# How It Works

1. The watcher starts and loads the Whisper model
2. It continuously monitors the `Input` folder
3. When a supported media file is detected:

   * The file is transcribed
   * Transcript text is saved as `.txt`
4. Files already transcribed are ignored automatically

# Installation

## 1. Install Python

Install Python 3.10+:

[Python Official Website](https://www.python.org/downloads/?utm_source=chatgpt.com)

---

## 2. Install FFmpeg

Whisper requires FFmpeg.

### Windows

Download FFmpeg:

[FFmpeg Official Website](https://ffmpeg.org/download.html?utm_source=chatgpt.com)

Add FFmpeg to your system PATH.

---

## 3. Install Dependencies

Open terminal or PowerShell:

```bash
pip install faster-whisper
```

Optional CPU optimization:

```bash
pip install ctranslate2
```

# Customization

## Use GPU Instead of CPU

If using an NVIDIA GPU:

```python
DEVICE = "cuda"
COMPUTE_TYPE = "float16"
```

---

## Faster / Smaller Models

```python
MODEL_SIZE = "tiny"
MODEL_SIZE = "base"
MODEL_SIZE = "small"
MODEL_SIZE = "medium"
MODEL_SIZE = "large-v3"
```

Tradeoff:

| Model    | Speed     | Accuracy |
| -------- | --------- | -------- |
| tiny     | Very Fast | Lower    |
| base     | Fast      | Moderate |
| small    | Balanced  | Good     |
| medium   | Slower    | Better   |
| large-v3 | Slowest   | Best     |

# Troubleshooting

## FFmpeg Not Found

Error:

```text
ffmpeg was not found
```

Fix:

* Install FFmpeg
* Add FFmpeg to PATH
* Restart terminal

---

## Slow Performance

If transcription is slow:

* Use smaller model (`small` or `base`)
* Enable GPU acceleration
* Upgrade to faster CPU/GPU

---

## File Not Transcribing

Ensure:

* File extension is supported
* File is not corrupted
* Output transcript does not already exist
